            <h2 class="text-3xl font-bold text-gray-900 mb-4 border-b pb-2">III. Memory Safety: Static-First ARC Model</h2>
            <p class="text-lg text-gray-700">
              Nebula achieves memory safety without a runtime Garbage Collector (GC) using **Static-First ARC Elision**. This is a hybrid approach where the compiler attempts **zero-cost** safety via static analysis (like Rust's borrow checker) but falls back to implicit **Atomic Reference Counting (ARC)** when lifetimes are ambiguous.
            </p>

            <h3 class="text-xl font-semibold text-gray-900 mt-6 mb-2">The Zero-Cost Tier (Performance Opt-In)</h3>
            <p class="text-base text-gray-700 mb-4">
              To guarantee zero memory overhead, the developer uses explicit keywords to guide the compiler:
            </p>
            <ul class="list-disc list-inside text-gray-700 pl-4 space-y-2">
              <li>**Exclusive Mutability (`mut`):** Asserts the function requires a unique reference. The compiler performs a **Borrow Check** and, if successful, **elides the ARC overhead.**</li>
              <li>**Move Semantics:** Transfers ownership of a resource, allowing the compiler to insert a direct, unconditional **`free()`** instruction.</li>
            </ul>

            <CodeBlock code={'// Zero-Cost: Exclusive Mutable Access (guaranteed by compiler)\npublic void Resize(mut Image img, int32 newSize) {\n    img.Size = newSize;\n}\n\n// Zero-Cost: Ownership Move\npublic Image ApplyFilter(Image img) { // \'img\' is consumed\n    return new Image { /* filtered data */ }; // Old \'img\' is freed\n}'} />

            <h3 class="text-xl font-semibold text-gray-900 mt-6 mb-2">The Runtime-Safe Tier (Abstraction Fallback)</h3>
            <p class="text-base text-gray-700 mb-4">
              When code is written in an abstracted, C#-like style where ownership is unclear (e.g., adding a reference to a global list), the compiler automatically and transparently inserts ARC instructions. This ensures memory safety at the cost of minimal runtime counter updates, abstracting memory complexity from the developer.
            </p>
            <CodeBlock code={'// High Abstraction: No explicit ownership syntax needed\n// Compiler implicitly inserts ARC ops.\nThreadQueue.enqueue(filteredImage); // ARC ensures the object lives until the queue is done with it.'} />

// src/components/Scroll.astro
<script>
    import LocomotiveScroll from "locomotive-scroll";
    import "locomotive-scroll/dist/locomotive-scroll.css";

    document.addEventListener("DOMContentLoaded", () => {
        const isMobile = window.matchMedia("(max-width: 768px)").matches;
        if (isMobile) return;

        const scrollContainer = document.querySelector(
            "[data-scroll-container]",
        ) as HTMLElement;

        if (!scrollContainer) {
            console.error(
                "Locomotive Scroll: Element with data-scroll-container not found.",
            );
            return;
        }

        const scroll = new LocomotiveScroll({
            el: scrollContainer,
            smooth: true,
            lerp: 0.07,
            multiplier: 1.5,
        });

        // Listen to the Locomotive Scroll 'scroll' event
        scroll.on("scroll", (obj) => {
            // Dispatch a custom event with the scroll data
            const event = new CustomEvent("loco-scroll", {
                detail: {
                    scrollPosition: obj.scroll.y,
                },
            });
            window.dispatchEvent(event);
        });

        window.addEventListener("beforeunload", () => {
            scroll.destroy();
        });
    });
</script>
